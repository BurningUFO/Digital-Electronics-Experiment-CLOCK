# CLOCK 到 Nexys A7 的首轮迁移说明

本轮只做旧 `repo_inspect_CLOCK` 工程到 `clock_amd` 的核心逻辑迁移，并补上 Nexys A7 必需的板级适配层。

## 迁移结果

- 旧工程全部核心 `.v` 文件已复制到 `clock_amd/clock_amd.srcs/sources_1/new/`
- `clock_amd.xpr` 已加入源码和约束文件
- 新增 Nexys A7 顶层 `clock_amd_top.v`
- 新增八位七段数码管扫描驱动 `nexys_seg_scan.v`
- 新增约束文件 `clock_amd/clock_amd.srcs/constrs_1/new/clock_amd.xdc`

## 本轮关键改动

1. 旧工程原来直接使用 `1kHz` 外部时钟输入
2. 迁移后改为统一使用板载 `100MHz`
3. 在顶层内生成 `1kHz` 时钟使能脉冲 `tick_1k`
4. 计时、按键、闹铃、倒计时等状态模块全部改成单时钟域工作
5. 七段显示改成适配 Nexys A7 的 `AN0~AN7 + CA~CG + DP` 动态扫描，且处理了低有效极性

## 当前按键/开关映射

- `BTNC`：旧工程 `QD` 统一执行键
- `CPU_RESETN`：低有效复位
- `SW0`：旧工程 `SWA`
- `SW1`：旧工程 `SWB`
- `SW2`：旧工程 `SWC`
- `SW3`：校时模式 `K1`
- `SW4`：闹铃模式 `K2`
- `SW5`：12/24 小时制模式 `K3`
- `SW6`：倒计时模式 `K4`
- `SW7`：作息提醒模式 `K5`

## 当前数码管显示布局

- `AN0`：秒个位 / 模式提示位 / 闹铃小数点提醒
- `AN1`：秒十位
- `AN2`：分个位
- `AN3`：分十位
- `AN4`：时个位
- `AN5`：时十位
- `AN6`：空白
- `AN7`：空白
