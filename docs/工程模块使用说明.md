# CLOCK 工程模块使用说明

本文档面向当前 `clock_amd` 工程，统一说明：

- 当前已经接入的新平台模块结构
- Nexys A7 100T 的按键、开关和数码管映射
- 各模式下的显示与交互规则
- 上板测试流程和记录项

本文件已经合并原 `工程模块使用说明.md` 与 `clock测试文档.md` 的内容，不再区分“说明文档”和“测试文档”。

## 1. 当前工程范围

当前主线工程位于 `clock_amd/`，已经完成从旧 Quartus/TEC-8 平台到 Vivado/Nexys A7 100T 的首轮迁移。

当前版本已经可用的功能：

1. 24 小时制正常走时
2. 校时功能
3. 闹铃设置与到点 `DP` 提醒
4. 倒计时设置、启动、暂停、到零停止
5. 模式优先级切换
6. Nexys A7 八位七段数码管动态扫描显示

当前仍为占位态、暂不计入正式功能通过项：

1. 12/24 小时制模式
2. 作息提醒模式

## 2. 当前模块结构

### 2.1 板级适配模块

| 文件 | 模块名 | 作用 |
| --- | --- | --- |
| `clock_amd_top.v` | `clock_amd_top` | Nexys A7 顶层，完成 `CLK100MHZ/CPU_RESETN/BTNC/SW0~SW7` 到内部时钟与控制接口的映射 |
| `nexys_seg_scan.v` | `nexys_seg_scan` | 将内部 6 位显示数据组织为 Nexys A7 八位低有效动态扫描输出 |

### 2.2 功能主线模块

| 文件 | 模块名 | 作用 |
| --- | --- | --- |
| `clock.v` | `clock` | 主线集成模块，连接所有功能模块 |
| `clk_ring.v` | `clk_ring` | 在 `tick_1k` 基础上生成 `1Hz` 计时脉冲 |
| `key_ctrl.v` | `key_ctrl` | 处理执行键消抖，并译码动作码 |
| `mode_ctrl.v` | `mode_ctrl` | 根据独立模式开关优先级实时生成 `mode_state` |
| `time_set_ctrl.v` | `time_set_ctrl` | 校时时切换“设置小时/设置分钟”，并生成加一脉冲 |
| `time_core.v` | `time_core` | 保存当前时间，负责正常走时和校时加一 |
| `alarm_ctrl.v` | `alarm_ctrl` | 保存闹铃时分与使能位，并完成当前时分实时比较 |
| `countdown_ctrl.v` | `countdown_ctrl` | 保存倒计时 `MM:SS`，负责设置、启动、暂停和到零停止 |
| `display_ctrl.v` | `display_ctrl` | 统一组织六位显示内容 |
| `seg_7.v` | `seg_7` | 4 位 BCD 到七段码译码 |

### 2.3 当前时序适配方式

旧平台直接提供 `1kHz` 时钟输入，当前平台使用板载 `100MHz`。

当前迁移版本采用：

1. `CLK100MHZ` 作为唯一主时钟
2. 在 `clock_amd_top.v` 中生成 `tick_1k`
3. 各功能状态模块统一采用“单时钟域 + 时钟使能”方式工作
4. `clk_ring.v` 再基于 `tick_1k` 生成 `1Hz` 计时脉冲

这样做的目的，是避免在 Vivado 中把普通分频寄存器直接作为新的慢时钟使用，降低派生时钟带来的时序和布线风险。

## 3. 当前板上输入映射

### 3.1 按键

| 板上信号 | 当前用途 | 说明 |
| --- | --- | --- |
| `CPU_RESETN` | 系统复位 | 低有效，按下时复位 |
| `BTNC` | 统一执行键 | 对应旧工程中的 `QD` |

### 3.2 开关

| 板上信号 | 当前用途 |
| --- | --- |
| `SW0` | `SWA` |
| `SW1` | `SWB` |
| `SW2` | `SWC` |
| `SW3` | 校时模式 |
| `SW4` | 闹铃模式 |
| `SW5` | 12/24 小时制模式占位 |
| `SW6` | 倒计时模式 |
| `SW7` | 作息提醒模式占位 |

### 3.3 动作码规则

动作码按 `SW2 SW1 SW0` 三位理解。

| `SW2 SW1 SW0` | `ctrl_sel` | 含义 |
| --- | --- | --- |
| `000` | `3'b000` | 当前无公共动作 |
| `001` | `3'b001` | 选择当前设置项 |
| `010` | `3'b010` | 当前数值加一 |
| `011` | `3'b011` | 确认 / 启停 / 开关 |
| `100` | `3'b100` | 倒计时模式下秒十位加一 |
| `101` | `3'b101` | 倒计时模式下秒个位加一 |
| `110` | `3'b110` | 当前无公共动作 |
| `111` | `3'b111` | 当前无公共动作 |

## 4. 模式优先级

若多个模式开关同时打开，优先级为：

`SW7 > SW6 > SW5 > SW4 > SW3`

对应模式如下：

| 优先级 | 条件 | `mode_state` | 当前模式 |
| --- | --- | --- | --- |
| 最高 | `SW7=1` | `3'b101` | 作息提醒模式占位 |
| 2 | `SW6=1` | `3'b100` | 倒计时模式 |
| 3 | `SW5=1` | `3'b011` | 12/24 小时制模式占位 |
| 4 | `SW4=1` | `3'b010` | 闹铃模式 |
| 5 | `SW3=1` | `3'b001` | 校时模式 |
| 默认 | `SW3~SW7` 全为 `0` | `3'b000` | 正常时钟模式 |

## 5. 数码管显示说明

当前工程使用 6 位显示时分秒，另外 2 位留空。

| 数码管位 | 当前内容 |
| --- | --- |
| `AN0` | 秒个位 |
| `AN1` | 秒十位 |
| `AN2` | 分个位 |
| `AN3` | 分十位 |
| `AN4` | 时个位 |
| `AN5` | 时十位 |
| `AN6` | 空白 |
| `AN7` | 空白 |

说明：

1. Nexys A7 数码管是共阳极、动态扫描、位选和段选都低有效
2. 当前工程已经在 `nexys_seg_scan.v` 中完成极性翻转和动态扫描
3. `DP` 当前复用为闹铃提醒输出

## 6. 当前功能与交互规则

### 6.1 正常时钟模式

进入条件：

- `SW3=0`
- `SW4=0`
- `SW5=0`
- `SW6=0`
- `SW7=0`

行为：

1. 复位后回到 `00:00:00`
2. 系统按 24 小时制正常走时
3. `23:59:59 -> 00:00:00`

### 6.2 校时模式

进入条件：

- `SW3=1`
- `SW4=0`
- `SW5=0`
- `SW6=0`
- `SW7=0`

显示规则：

1. 秒个位显示字段提示
2. `1` 表示当前设置小时
3. `2` 表示当前设置分钟

操作规则：

1. `001 + BTNC`：在“设置小时 / 设置分钟”之间切换
2. `010 + BTNC`：当前字段加一
3. 小时加一规则：`23 -> 00`
4. 分钟加一规则：`59 -> 00`
5. 退出校时模式后再次进入，默认回到“设置小时”

### 6.3 闹铃模式

进入条件：

- `SW4=1`
- `SW5=0`
- `SW6=0`
- `SW7=0`

显示规则：

闹铃模式显示为 `HH:MM:E0`

1. 时十、时个：闹铃小时
2. 分十、分个：闹铃分钟
3. 秒十：闹铃使能位 `E`
4. 秒个：固定显示 `0`

操作规则：

1. `001 + BTNC`：闹铃小时加一
2. `010 + BTNC`：闹铃分钟加一
3. `011 + BTNC`：闹铃开关翻转
4. 小时回卷：`23 -> 00`
5. 分钟回卷：`59 -> 00`

提醒规则：

1. 当闹铃开启且当前时分等于设定闹铃时分时，`DP` 点亮
2. 当前实现中，`DP` 会在整个匹配分钟内持续点亮
3. `DP` 提醒不依赖是否停留在闹铃模式

### 6.4 倒计时模式

进入条件：

- `SW6=1`
- `SW7=0`

显示规则：

倒计时模式显示为 `00:MM:SS`

1. 时十、时个固定显示 `0`
2. 分十、分个显示倒计时分钟
3. 秒十、秒个显示倒计时秒

操作规则：

1. `001 + BTNC`：分钟十位加一
2. `010 + BTNC`：分钟个位加一
3. `100 + BTNC`：秒十位加一
4. `101 + BTNC`：秒个位加一
5. `011 + BTNC`：开始 / 暂停
6. 秒十位范围限制为 `0~5`
7. 当前值不为 `00:00` 时才能启动
8. 倒计时减到 `00:00` 时自动停止

### 6.5 占位模式

`SW5` 和 `SW7` 对应的两个模式当前仅做模式位占位：

1. 12/24 小时制模式未接入实际转换逻辑
2. 作息提醒模式未接入实际提醒逻辑
3. 当前测试只要求确认进入后系统不死机、显示不完全异常

## 7. 上板测试流程

建议按以下顺序测试：

1. 基础显示与复位
2. 正常时钟模式
3. 校时模式
4. 闹铃模式
5. 倒计时模式
6. 占位模式检查
7. 模式优先级检查
8. 七段显示专项检查

## 8. 详细测试步骤

### 8.1 基础显示与复位测试

步骤：

1. 下载 bitstream 后，保持 `SW3~SW7=0`
2. 按一次 `CPU_RESETN`
3. 观察数码管显示
4. 持续观察几秒

预期结果：

1. 显示回到 `00:00:00`
2. 秒正常递增
3. 无全灭、全亮、严重鬼影或无规律闪烁

记录项：

- [ ] 复位后显示 `00:00:00`
- [ ] 秒正常递增
- [ ] 位序正确
- [ ] 段码正确
- [ ] 无明显闪烁/鬼影异常

### 8.2 正常时钟模式测试

步骤：

1. 保持所有模式开关关闭
2. 按一次 `CPU_RESETN`
3. 观察秒递增
4. 继续观察分、时进位

预期结果：

1. `00:00:00 -> 00:00:01 -> 00:00:02 ...`
2. `00:00:59 -> 00:01:00`
3. `00:59:59 -> 01:00:00`
4. `23:59:59 -> 00:00:00`

记录项：

- [ ] 秒进位正确
- [ ] 分进位正确
- [ ] 小时进位正确
- [ ] `23:59:59` 回卷正确

### 8.3 校时模式测试

步骤：

1. 打开 `SW3`
2. 观察秒个位是否显示 `1`
3. 把动作码设为 `001`
4. 按一次 `BTNC`
5. 观察秒个位是否变成 `2`
6. 把动作码设为 `010`
7. 按一次 `BTNC`
8. 观察分钟是否加一
9. 再把动作码设为 `001`
10. 按一次 `BTNC`
11. 观察提示是否回到 `1`
12. 保持动作码 `010`
13. 按一次 `BTNC`
14. 观察小时是否加一
15. 退出再进入，确认默认回到小时设置

记录项：

- [ ] 默认提示为 `1`
- [ ] `001 + BTNC` 能切到提示 `2`
- [ ] `010 + BTNC` 能使分钟加一
- [ ] 切回后提示重新变成 `1`
- [ ] `010 + BTNC` 能使小时加一
- [ ] 小时回卷正确
- [ ] 分钟回卷正确
- [ ] 再次进入时默认回到小时设置

### 8.4 闹铃模式测试

步骤：

1. 打开 `SW4`
2. 观察当前显示是否为 `HH:MM:E0` 风格
3. 用 `001 + BTNC` 测试闹铃小时加一
4. 用 `010 + BTNC` 测试闹铃分钟加一
5. 用 `011 + BTNC` 测试闹铃开关在 `0/1` 间切换

到点提醒测试：

1. 记下当前正常时间
2. 进入闹铃模式
3. 把闹铃设置到“当前时间的下一分钟”
4. 打开闹铃
5. 回正常模式等待
6. 观察时间到达后 `DP` 是否点亮

记录项：

- [ ] 闹铃小时可设置
- [ ] 闹铃分钟可设置
- [ ] 闹铃开关可切换
- [ ] 小时回卷正确
- [ ] 分钟回卷正确
- [ ] 到点后 `DP` 点亮
- [ ] `DP` 不影响数字主体显示

### 8.5 倒计时模式测试

建议先从短时间开始，例如 `00:00:05` 或 `00:00:10`。

步骤：

1. 打开 `SW6`
2. 用 `001 + BTNC` 测试分钟十位加一
3. 用 `010 + BTNC` 测试分钟个位加一
4. 用 `100 + BTNC` 测试秒十位加一
5. 用 `101 + BTNC` 测试秒个位加一
6. 用 `011 + BTNC` 启动倒计时
7. 再次 `011 + BTNC` 测试暂停
8. 再次 `011 + BTNC` 测试恢复
9. 等待倒计时减到 `00:00`

记录项：

- [ ] 分钟十位可设置
- [ ] 分钟个位可设置
- [ ] 秒十位可设置且范围正确
- [ ] 秒个位可设置
- [ ] 能启动
- [ ] 能暂停
- [ ] 能恢复
- [ ] 到零自动停止

### 8.6 占位模式检查

检查目标：

1. 打开 `SW5` 后系统不死机
2. 打开 `SW7` 后系统不死机
3. 显示不出现完全异常的无规则闪烁

记录项：

- [ ] `SW5` 占位模式进入正常
- [ ] `SW7` 占位模式进入正常
- [ ] 两者当前都不计入正式功能通过项

### 8.7 模式优先级测试

步骤：

1. 先打开 `SW3`
2. 再打开 `SW4`
3. 确认进入闹铃模式而不是校时模式
4. 在此基础上再打开 `SW6`
5. 确认进入倒计时模式
6. 在此基础上再打开 `SW7`
7. 确认进入作息提醒占位态

记录项：

- [ ] `SW4` 优先于 `SW3`
- [ ] `SW6` 优先于 `SW5/SW4/SW3`
- [ ] `SW7` 优先于全部其他模式

### 8.8 七段显示专项检查

检查项：

1. 数字 `0~9` 是否都能正常显示
2. 是否存在固定坏段
3. 是否存在固定常亮段
4. 切换模式时显示内容是否符合预期
5. 动态扫描稳定性是否可接受

记录项：

- [ ] `0~9` 段码正常
- [ ] 无固定坏段现象
- [ ] 无固定常亮段现象
- [ ] 模式切换显示符合预期
- [ ] 扫描显示稳定
