| `k2_mode_alarm` | `PIN_80` | 模式选择输入 2 | 闹铃模式开关 |

## 2. 显示相关引脚分配

本工程一共驱动 6 个数码管位置：

- `LG1` 输出七段段码，用来显示“秒个位”
- `LG2`~`LG6` 输出 BCD 码，分别显示“秒十位、分个位、分十位、时个位、时十位”

### 2.1 LG1 段码输出

`sec_unit_seg[7:0]` 对应数码管 `LG1` 的 `a b c d e f g dp`。

| 段信号 | 工程信号 | CPLD 管脚 |
| --- | --- | --- |
| a | `sec_unit_seg[0]` | `PIN_44` |
| b | `sec_unit_seg[1]` | `PIN_45` |
| c | `sec_unit_seg[2]` | `PIN_46` |
| d | `sec_unit_seg[3]` | `PIN_48` |
| e | `sec_unit_seg[4]` | `PIN_49` |
| f | `sec_unit_seg[5]` | `PIN_50` |
| g | `sec_unit_seg[6]` | `PIN_51` |
| dp | `sec_unit_seg[7]` | `PIN_52` |

### 2.2 LG2 到 LG6 的 BCD 输出

BCD 位序统一为：

- `A -> bit0`，权值 `1`
- `B -> bit1`，权值 `2`
- `C -> bit2`，权值 `4`
- `D -> bit3`，权值 `8`

| 显示位 | 显示内容 | 工程信号 | CPLD 管脚 |
| --- | --- | --- | --- |
| `LG2` | 秒十位 | `sec_ten_bcd[0]~[3]` | `PIN_37, 39, 40, 41` |
| `LG3` | 分个位 | `min_unit_bcd[0]~[3]` | `PIN_35, 36, 17, 18` |
| `LG4` | 分十位 | `min_ten_bcd[0]~[3]` | `PIN_30, 31, 33, 34` |
| `LG5` | 时个位 | `hour_unit_bcd[0]~[3]` | `PIN_25, 27, 28, 29` |
| `LG6` | 时十位 | `hour_ten_bcd[0]~[3]` | `PIN_20, 21, 22, 24` |

## 4. 使用说明

- 本工程当前使用了复位、时钟、`QD`、`SWA/SWB/SWC` 以及五路独立模式选择输入。
- `K1~K5` 对应的模式输入当前锁定为 `PIN_81`、`PIN_80`、`PIN_79`、`PIN_77`、`PIN_76`。
- 模式优先级为 `K5 > K4 > K3 > K2 > K1`，全部关闭时回到正常时钟模式。
- `LG1` 采用七段段码直驱，因此 [`clock.v`](/C:/Users/YOUNG/Desktop/workplace/2025_2026_2/SDSY/CLOCK/clock.v) 中有 `seg_7` 译码模块。
- `LG2` 到 `LG6` 直接输出 BCD 码，不在 CPLD 内再做七段译码。
- 若后续需要改成别的时钟源，可以优先调整 `clk_1k` 对应的引脚约束，而不用改显示部分引脚。
